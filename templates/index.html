Symmetric algorithms are used to improve security in applications. The one that I made for this application is but a demonstration of one simple but effective algorithm that can enhance security. The user would input a message for encryption. This program would then look through each character of that string and translate it to Unicode.

The Unicode representation of that character (which would be a number) would then be added by 5 and then converted back into a character. This would then be repeated for each character of that string and then all of them would be put in order and assigned to one single variable of type string. That variable would contain the encryption of that message.

The decryption portion of the algorithm is doing the exact opposite. For each character, you convert it into its Unicode representation, subtract it by 5, and then convert it back into a character. Putting all of these characters in order and assigned to one variable of type string, the message would be decrypted and printed in its original format.
